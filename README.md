**CryptoPulse** es una Single Page Application desarrollada con React que permite explorar información actualizada sobre criptomonedas utilizando una API pública.

El proyecto fue creado con fines educativos y de portfolio, poniendo foco en:

- Navegación con React Router
- Manejo de estado global con Context API
- Creación y uso de custom hooks
- Consumo de APIs REST
- Formularios, estados de carga y manejo de errores

La aplicación prioriza una arquitectura clara, componentes reutilizables y una experiencia de usuario simple e intuitiva.
